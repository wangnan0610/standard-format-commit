# standard-format-commit
团队提交的commit格式保持一致

## 初始化

1. yarn add standard-format-commit

2. ./node_modules/.bin/sfc-init
  
3. 初始化成功以后会有提示

4. 照着提示修改package.json添加如下scripts("commit": "node_modules/.bin/sfc-commit")

5. 基本可以使用了，当然你也可以根据自己的需求修改配置

## 使用

npm run commit(是不是非常简单)
